Fortnite Shoe Finder - Roboflow Project

🎮 What is Fortnite?

Fortnite is a globally popular online multiplayer game known for its fast-paced action, colorful graphics, and iconic character skins. With millions of active players and countless in-game cosmetics, it’s become a cultural phenomenon.

👟 Project Summary

This project helps sneakerheads and casual Fortnite players alike identify real-life shoes featured in the game and find them online. While some in-game shoes are named, many don’t match their retail names, making them hard to find. For example:

  • “Jordan 11 University Blue” (game) = Jordan 11 Retro Pantone (real life)
  • “Jordan 11 Black/Gym Red” (game) = Jordan 11 Playoffs Bred (real life)

This tool bridges that gap using computer vision via Roboflow.
